# 前言

Yii 是一个高性能、基于包的 PHP 框架，用于开发现代应用程序。
Yii（发音为 `Yee` 或 `[ji:]`）这个名字在中文中意味着"简单和不断演进"。
您也可以将其视为 **Yes It Is!** 的缩写！

Yii 是一个通用的 Web 编程框架。
您可以使用它来开发各种使用 PHP 的 Web 应用程序。
由于其架构和复杂的缓存支持，它特别适合开发大型应用程序，如门户、内容管理系统、
电子商务、RESTful API 等。

凭借全面的文档和热情的用户社区，Yii 从长远来看可以显著减少您的开发时间。

## 本书内容

如果您熟悉 Yii3、构建 Yii 应用程序并阅读了官方 Yii3 指南，本书就适合您。
它涵盖了基本重要的开发概念、应用程序架构方法、将第三方服务与 Yii3 集成等。

本书由来自 Yii 专家的单个食谱组成，您可以将其应用到您的应用程序中。
这些内容按主题分类，但您可以按任何顺序阅读它们，因为它们之间没有依赖关系。

## 前置条件

- 您应该已安装 Yii3。
- 您应该熟悉框架基础和官方指南。
