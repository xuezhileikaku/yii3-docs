# Yii3 社区食谱

Yii3 社区食谱是一本开源书籍，包含了关于 [Yii3](https://www.yiiframework.com/) PHP 框架的技巧和窍门。

- Yii 社区创建这本食谱。
- Yii 核心团队成员进行策划和编辑。

欢迎提交您的文章。团队成员将对其进行评审，给予反馈，并以最佳方式合并。

本书遵循 [Yii 文档条款](https://www.yiiframework.com/license#docs)。
