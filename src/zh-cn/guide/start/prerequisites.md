# 你需要了解什么？

Yii 的学习曲线并不像其他 PHP 框架那样陡峭，但仍然有一些你应该在开始使用 Yii 之前学习的内容。

## PHP

Yii 是一个 PHP 框架，因此请确保你[阅读并理解语言参考](https://www.php.net/manual/en/langref.php)。

## 面向对象编程

你需要对面向对象编程有基本的了解。如果你不熟悉它，可以查看许多可用的教程之一，例如 [tuts+ 的教程](https://code.tutsplus.com/tutorials/object-oriented-php-for-beginners--net-12762)。

当你使用 Yii 开发时，你以面向对象的方式编写代码，因此请确保你熟悉 [PHP OOP 支持](https://www.php.net/manual/en/language.oop5.php)。

请注意，你的应用程序越复杂，你应该学习的高级 OOP 概念就越多，以便成功地管理这种复杂性。

## 命令行和 Composer

Yii 广泛使用事实上的标准 PHP 包管理器 [Composer](https://getcomposer.org)，因此请确保你阅读并理解其[指南](https://getcomposer.org/doc/01-basic-usage.md)。如果你不熟悉使用命令行，那么是时候开始尝试了。一旦你学会了基础知识，你就永远不想在没有它的情况下工作了。

## HTTP

由于 Yii 是一个 Web 框架，而 Web 很大程度上使用 HTTP，因此[更多地了解它](https://developer.mozilla.org/en-US/docs/Web/HTTP)是一个好主意。

## Docker

默认应用程序模板利用 Docker，因此我们建议你[阅读并理解这些概念](https://docs.docker.com/get-started/)。

此外，你将受益于熟悉 [twelve-factor app](https://12factor.net/) 原则。
