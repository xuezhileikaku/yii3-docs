# 缓存

缓存是一种廉价且有效的应用程序性能提升方式。
通过将相对静态的数据存储在缓存中，并在请求时从缓存中提供数据，
应用程序节省了每次都需要从头生成数据的时间。

缓存可以在应用程序的不同级别和位置进行。在服务器端，在较低级别，
缓存可用于存储基本数据，例如从数据库获取的最新文章信息列表；
在较高级别，缓存可用于存储 Web 页面的片段或整个页面，例如最新文章的渲染结果。
在客户端，你可以使用 HTTP 缓存将最近访问的页面内容保留在浏览器缓存中。

Yii 支持所有这些缓存机制：

* [数据缓存](data.md)
* [片段缓存](fragment.md)
* [页面缓存](page.md)
* [HTTP 缓存](http.md)
